# JOBLESS - How fast is AI replacing human jobs?

A data-driven interactive website showing AI's replacement of human jobs, with job risk rankings, global impact data, and future predictions.

## Features

- **Real-time Statistics**: MIT and McKinsey data on AI job replacement
- **Interactive Timeline**: Key AI milestones from 2020 to 2040+
- **Risk Assessment**: High-risk job rankings by industry
- **Layoff Tracker**: Real company case studies
- **Survival Index**: Calculate your AI replacement risk
- **Bilingual**: English and Chinese language support

## Tech Stack

- Next.js 16.1.6 with App Router
- React 19.2.3
- Tailwind CSS v4
- Framer Motion (animations)
- Recharts (data visualization)
- Playwright (E2E testing)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Run Tests

```bash
npm run test
```
